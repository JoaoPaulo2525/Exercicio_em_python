Meu Exercícios de Python

Este repositório contém meus códigos de estudo em Python, incluindo desafios de lógica, manipulação de strings e cálculos matemáticos.

## 🚀 O que este projeto faz?
Atualmente, o script principal (`Ex3.py`) realiza as seguintes funções:
* Recebe o nome do usuário e calcula o número de caracteres.
* Calcula a raiz quadrada de um número informado usando a biblioteca `math`.
* Demonstra o uso de listas, soma de elementos (`sum`) e contagem de itens (`len`).

## 🛠️ Tecnologias Utilizadas
* **Python 3.12**
* **VS Code** (Ambiente de desenvolvimento)

## 📁 Estrutura de Arquivos
* `Ex3.py`: Exercício sobre funções matemáticas e listas.
